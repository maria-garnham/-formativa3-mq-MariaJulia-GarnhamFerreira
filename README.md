# formativa3-mq-MariaJulia-GarnhamFerreira
Repositório criado para a atividade formativa 3 de Front-End


💻 GalaxyTours - Media Query
Este é um projeto desenvolvido para a avaliação formativa de Front-End, focado em tornar o site da GalaxyTours totalmente responsivo, garantindo uma boa experiência de uso em tablets e smartphones através do uso de Media Queries

🛠️ Tecnologias Utilizadas
HTML5 (Semântica e Acessibilidade)
CSS3 (Estilização e Feedback Visual)
📁 Estrutura do Projeto
O projeto foi organizado seguindo boas práticas de mercado: 
  / ├── index.html 
      └── src/ 
        ├── styles/ (Estilos da página)
        ├── scripts/ (Lógica de interação) 
        └── images/ (Ativos visuais)

🚀 Funcionalidades Implementadas
Responsividade Completa: Adaptação do layout para diferentes tamanhos de tela (desktop, tablet e mobile)
Grid Adaptativo: Alteração dinâmica do número de colunas dos pacotes espaciais conforme o dispositivo
Layout Flexível: Ajuste do cabeçalho para melhor leitura em telas menores
Experiência Mobile Otimizada: Ocultação de elementos desnecessários em dispositivos móveis.
